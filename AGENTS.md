# Project Guardrails

## Live Data Source

- 현재 카페24 공용 저장소/API의 데이터를 기준으로 유지한다.
- 엑셀 파일, 기본값, 샘플 데이터, 로컬 저장소 값으로 현재 카페24 공용 저장소의 데이터를 절대 덮어쓰지 않는다.
- 인원명단, 반 시간표, 강의실, 클리닉 회차, 시험 성적, 출석, To-Do, 교재DB처럼 사용자가 입력한 운영 데이터는 서버의 현재 값을 우선한다.
- 마이그레이션, seed, fallback, import, 캐시 복구 코드를 만들거나 수정할 때도 기존 서버 데이터를 보존하는지 먼저 확인한다.
- 사용자가 명시적으로 요청하지 않는 한 과거 엑셀 자료나 테스트 데이터를 재적용하지 않는다.

## Clinic Listening Uploads

- 듣기 음성 파일과 듣기 교재/해설 파일의 실제 파일 용량 제한은 100MB다.
- Base64 전송 여유분 때문에 Express JSON과 Nginx 요청 제한은 160MB로 둔다.

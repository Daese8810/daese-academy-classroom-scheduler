# Firebase는 어떻게 쓰면 좋을까?

이번 1차 실배포는 Cafe24 VPS + PostgreSQL 구조를 권장합니다.

그 이유:
- 한국어 아이디/비밀번호 기반 교사 계정 관리가 더 단순합니다.
- 강의실 시간 겹침 금지를 PostgreSQL 제약으로 강하게 보장할 수 있습니다.
- 반복 예약, 관리자 차단, 수정/취소 로직을 서버에서 일관되게 처리하기 쉽습니다.

Firebase는 아래 용도로는 잘 맞습니다.
- 추후 모바일 알림
- 통계/접속 로그 연동
- 별도 미리보기용 정적 페이지
- 다른 도메인으로 간단한 랜딩 페이지 운영
